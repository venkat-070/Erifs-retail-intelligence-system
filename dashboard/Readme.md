---Instructions for Best Experience:
1. **Desktop App:** Please open this file in the **Excel Desktop App**. Web-based viewers (like Google Sheets or Excel Online) may not support the advanced Pivot Cache or the **81% accuracy forecasting formulas**.
2. **Enable Content:** If prompted, click "Enable Editing" or "Enable Content" to allow the forecasting models to initialize.
3. **Data Source:** To refresh the background data, ensure you have downloaded the raw dataset mentioned in the
4. [main project README](../README.md).
